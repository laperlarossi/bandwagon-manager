# bwh_controller

- python => 3.5
- pyqt => 5.6

# Features

## Status
- VPS RAM/SWAP/bandwidth/disk_usage/OS status
- ssh port 
- IP address

## Control
- start/stop/restart/kill VPS
- basic root shell

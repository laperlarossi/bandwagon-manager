# 搬瓦工VPS控制器

- python版本 => 3.5
- pyqt版本 => 5.6

# 功能

## 查看VPS状态
- VPS 内存/交换分区/流量/存储/系统 情况
- ssh 端口 
- IP 地址

## 基本控制
- 启动/停止/重启/强制关机 
- 基础root shell
